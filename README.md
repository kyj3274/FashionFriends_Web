# FashionFriends
실시간 날씨와 내 옷장 데이터를 조합해 코디를 제안하는 웹 서비스입니다. 매일 아침 "오늘 뭐 입지?" 고민하는 시간을 줄여주고, 내 착장을 기록하며 다른 사람들과 스타일을 공유할 수 있습니다.

## 주요 기능
1. AI 옷장 관리 & 분석
   자동 등록: 옷 사진을 올리면 AI가 카테고리와 색상을 분석해서 저장합니다.

    컬러 팔레트: 내 옷장에 어떤 색상의 옷이 많은지 비율을 차트로 보여줍니다.

2. 날씨 맞춤형 AI 추천
   실시간 날씨: 위치 기반 API로 현재 기온과 상태(눈, 비, 강풍 등)를 가져옵니다.

    코디 제안: "비가 오니 젖지 않는 신발을 신으세요" 같은 구체적인 이유와 함께 내 옷장에서 최적의 조합을 골라줍니다.

3. OOTD 기록 (달력 & 피드)
   착장 기록: 그날 입은 옷을 사진이나 아이템 조합으로 달력에 저장합니다.

    소셜 공유: 내 코디를 공개 피드에 올려 '좋아요'를 받거나 다른 사용자를 팔로우(Ref) 할 수 있습니다.

##  기술 스택
Frontend: Vanilla JavaScript, CSS3, HTML5

Backend: Java, Spring Boot

Database: MySQL

API: OpenWeatherMap API, OpenAI/Groq API
